# LoveDistance-Game
這是一個關於兩對情侶為了獲得愛神的祝福互相競爭的遊戲<br>
總共四名玩家分成兩組
誰先跳到最上面的平台並支撐到愛情能量條集滿就勝利<br>
場上會有道具可進行撿取<br>
玩家身上的線象徵著愛情的痕跡
既不能離得太遠也不能靠得太近
代表愛情的矛盾<br>
